# IRP Label Proofreading API

AI-powered pharmaceutical label proofreading system built with FastAPI and OpenAI.

## Features

- Upload Redline PDFs
- Upload LRF reference forms
- Upload Final Labels
- AI-powered label verification
- PDF analysis
- Report generation

## Tech Stack

- FastAPI
- OpenAI
- Python
- PDF Processing
- REST APIs

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
